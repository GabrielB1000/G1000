Neste documento explicaremos as tags aprendidas até agora no curso

As tags h1 até h6 servem para imediatamente comunicar o assunto de cada segmento em uma página, h1 possuindo a maior fonte e h6 e em diante a menor.

Aqui estamos usando h2, pode-se ver que possui uma fonte menor que a primeira linha, ela mesma sendo o h1

Além disso, h7 e diante existem e podem ser utilizadas mesmo não possuindo diferenças visíveis de h6

Algumas das outras tags neste documento são

  • mark, que marca o texto dentro da tag
  • small, que diminui a fonte do texto dentro da tag
  • u, que adiciona uma sublinha/underline ao texto dentro da tag
  • strong, que aplica negrito à fonte do texto dentro da tag
  • sub altera o posicionamento do texto dentro da tag e o seu tamanho, sendo movido para baixo
  • sup funciona de maneira quase identica à tag sub, exceto que move para cima
  • blockquote
    é uma tag usada para parafrasear informação externa

Outras tags não vistas no curso até agora porém sugeridas neste projeto, listadas com a tag ol são

  1. font, que foi utilizada em HTML 4 para especificar o tipo de fonte, tamanho de fonte e cor do texto, porém não é compatível com HTML5
  2. del define um texto que foi deletado, é visível porém sobrelinhado
  3. abbr define uma abreviação ou acrônimo, como WWW
